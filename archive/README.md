# Archive

Este directorio contiene código y scripts obsoletos del proyecto BotCompressor.

## Directorios

### old_versions/
Contiene versiones antiguas del código del bot:

- `bot_original.py` - Versión monolítica original del bot antes de la refactorización SOLID

### old_scripts/
Contiene scripts de inicio alternativos o descontinuados:

- `start_bot_fixed.sh` - Script alternativo de inicio con funcionalidad similar a start_bot.sh
- `auto_start_bot.sh` - Script para auto-reinicio con manejo de FLOOD_WAIT

## Notas

- **Script actual de inicio**: `start_bot.sh` (en la raíz del proyecto)
- Estos archivos se mantienen solo para referencia histórica
- No deben usarse en producción
- Si se necesita alguna funcionalidad específica de estos scripts, considere integrarla en `start_bot.sh`
